Detailed Description

Track animated sticker class.

Track animation sticker is a landscaping effect used in video editing to produce animated effects. Users can add and remove animated stickers from the track, as well as adjust various properties of the stickers through various APIs, such as position, size, display time, and so on.
Note: The in-point and out-point units of the animated sticker on the Track are both in microseconds.

Warning
In the NvsTrackAnimatedSticker class, all public APIs are used in the UI thread! ! !
Since
2.20.0
