枚举值

◆ NvsVideoFramePixelFormat

enum NvsVideoFramePixelFormat

枚举值
NvsPixelFormat_Nv12

NvsPixelFormat_YUV420 

YUV 4:2:0 12bpp ( 3通道, 一个亮度通道, 另两个为U分量和V分量通道, 所有通道都是连续的 )

NvsPixelFormat_BGRA 

BGRA

NvsPixelFormat_RGBA 

RGBA

NvsPixelFormat_P010 

YUV 4:2:0 24bpp ( 2通道, 一个通道是连续的亮度通道, 另一通道为VU分量交错 每个通道占用16位，高10位有效)
